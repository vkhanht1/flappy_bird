# 🐦 Flappy Bird (Pygame)

## 📌 Description

This is a simple **Flappy Bird clone** built using **Python** and the **Pygame** library.
The game recreates the classic mechanics where the player controls a bird that must fly through gaps between moving pipes without colliding.

The project is designed as a beginner-friendly game to practice:

* Game loops
* Collision detection
* Basic physics (gravity and movement)
* Event handling in Pygame

---

## 🎮 Features

* Smooth bird movement with gravity
* Randomly generated pipes
* Score tracking system
* Collision detection (pipes, ground, and ceiling)
* Game over screen with restart option
* Background and sprite rendering

---

## 🕹️ Controls

* **SPACE** → Make the bird jump
* **SPACE (after Game Over)** → Restart the game
* **Close Window** → Exit the game

---

## 🧠 How It Works

* The bird continuously falls due to gravity.
* Pressing SPACE applies an upward force.
* Pipes move from right to left at a constant speed.
* Each time the bird passes a pipe, the score increases.
* Collision with pipes or boundaries ends the game.
